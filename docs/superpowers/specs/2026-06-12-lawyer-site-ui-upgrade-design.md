# 律师网站UI升级设计规格

## 1. 概述

**目标：** 将现有律师个人宣传网站升级为现代简约风格，提升视觉品质和用户体验。

**范围：** 纯前端UI改进，不涉及功能变更。

**技术栈：** 单文件HTML，内嵌CSS和JavaScript。

## 2. 设计决策

### 2.1 配色方案

**原有配色：**
- 主色：深海军蓝 `--navy: #0c1a2e`
- 强调色：金色 `--gold: #c9a96e`

**新配色方案：黑白+金色**
- 主色：纯黑 `--black: #000000`
- 辅助色：深灰 `--dark: #1a1a1a`
- 背景色：纯白 `--white: #ffffff`
- 浅灰背景：`--light: #f8f8f8`
- 强调色：金色 `--gold: #c9a96e`
- 文字色：深灰 `--text: #333333`
- 次要文字：中灰 `--gray: #666666`

### 2.2 动画效果

#### 视差滚动
- Hero区域背景图片产生视差效果
- 滚动时各元素以不同速度移动，增加层次感

#### 微交互动画
- 卡片悬停：精致的阴影提升和轻微位移
- 按钮悬停：颜色渐变和阴影变化
- 按钮点击：波纹扩散效果
- 图标悬停：轻微放大和颜色变化

#### 文字动画
- 标题：淡入+上移动画
- 正文：逐段淡入
- 数字：计数动画（已有）

### 2.3 排版优化

#### 间距
- Section间距：从120px增加到140px
- 内边距：从6%增加到8%
- 卡片内边距：增加20%

#### 文字层次
- 主标题：`clamp(36px, 5vw, 56px)`
- 副标题：`clamp(20px, 3vw, 28px)`
- 正文：16px，行高1.9
- 说明文字：14px，颜色更浅

#### 装饰元素
- 分割线：更细，金色渐变
- 装饰边框：更精致的金色边框
- 背景纹理：微妙的几何图案

## 3. 组件改进

### 3.1 导航栏
- 背景：纯黑，半透明毛玻璃效果
- 高度：从72px优化到68px
- 悬停效果：金色下划线动画

### 3.2 Hero区域
- 背景：视差滚动效果
- 标题：更大的字号，更强的视觉冲击
- 按钮：更精致的样式和动效

### 3.3 关于我
- 照片区域：更精致的边框和阴影
- 统计数字：金色渐变效果
- 文字：更好的层次感

### 3.4 专业领域卡片
- 背景：纯白，精致阴影
- 悬停：阴影提升，边框金色渐变
- 图标：更精致的样式

### 3.5 客户评价
- 卡片：更精致的边框和阴影
- 星级：金色渐变
- 引号：更明显的装饰效果

### 3.6 联系表单
- 输入框：更精致的边框和聚焦效果
- 按钮：金色渐变，波纹点击效果

## 4. 响应式设计

### 桌面端（>1024px）
- 完整视差效果
- 所有动画效果
- 最大视觉冲击

### 平板端（768px-1024px）
- 简化视差效果
- 保留主要动画
- 优化触摸交互

### 移动端（<768px）
- 禁用视差效果（性能考虑）
- 简化动画
- 优化触摸目标大小

## 5. 性能考虑

### 动画优化
- 使用CSS transform和opacity进行动画
- 避免触发重排的属性
- 使用requestAnimationFrame优化JavaScript动画

### 图片优化
- 保持现有图片尺寸
- 使用loading="lazy"延迟加载非关键图片

## 6. 实现优先级

1. 配色方案更新
2. 排版和间距优化
3. 微交互动画
4. 视差滚动效果
5. 文字动画
6. 响应式优化

## 7. 验收标准

- [ ] 配色方案符合黑白+金色设计
- [ ] 所有动画效果流畅运行
- [ ] 移动端体验良好
- [ ] 页面加载性能不受影响
- [ ] 所有原有功能正常工作
